    Whether an iterator is equal with the iterator.

    Compare two iterators and returns whether they are equal or not.

    Note

    Iterator's equal_to() only compare souce container and index number.

    Although elements in a pair, key and value are equal_to, if the source map or index number is different, then the equal_to() will return false. If you want to compare the elements of a pair, compare them directly by yourself.
