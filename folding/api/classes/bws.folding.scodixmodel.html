Class ScodixModel

스코딕스 모형.

  1. 작업비: 25,000 원 / 1,000 면
  2. 동판비 (도장): 200 원 / cm2
  3. 박비 (잉크): 1,300 원 / m2

비고

  • 양/단면 선택 가능
  • 여유 용지: 각 판면당 30 매
  • 비확정 - 할증률

    스코딕스 공정이 적용되는 면수에 따라 스코딕스 가격에 할증률이 적용된다. 할증률은 출력면수에 반비례하여 스코딕스 공정이 적용되는 면수가 적으면 적을수록 할증률은 높으며, 면수가 많을수록 할증률은 작아진다. 이 할증률을 실제로 적용할 지 어떨 지는 아직 확정되지 않았다.

    추후 할증률을 사용하기로 결정되거든, extra_charge_map 에 달린 주석을 지우고, 스코딕스 수량을 키값으로, extra_charge_map.upper_bound() 함수를 호출하여 할증률 값을 받은 후, 이를 최종 가격에 곱해주면 된다.

    현 공정을 적용함에 있어 필요한 여분지, 여분지의 총 면적을 계산한다.

    각 공정마다 조판면을 구성함에 있어 각기 필요한 여분지 매수가 있다. 이렇게 필요한 모든 여분지의 전체 면적을 계산한다. 보통은 각 조판면의 면적에 대하여 여분지 매수를 곱하고, 이들을 합하여 총 면적을 구한다. OptionalModel.compute_spare_area 에서는, 이 보통의 경우를 계산한다.

    • 라미네이팅 코팅은, 각 조판면마다 여분용지가 필요한 게 아니고, 사용케 되는 용지 크기에 따라 여분 용지를 필요로 하게 된다. 때문에, LaminatingModel 은 별도의 여분용지면적 계산 메소드가 따로이 정의되어있다.
