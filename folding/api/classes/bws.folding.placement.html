Class Placement

조판면 배치 정보.

Placement 는 개별 공정 Process 에 소속되는, 조판면 배치 정보를 가진 Entity 및 Calculator 클래스이다.

현 조판면이 각 페이지를 어떻게 배치하는 지 (row_count, col_count, image_count), 이러한 조판면 배치가 몇 회 반복되는 지 (face_count) 및 어떠한 분할이 행해지는 지 (inversecount, plastecoiunt) 등의 정보를 가진다.

또한 Placment 는, 현 조판면 내에 각 페이지가 어떻게 배치되는 지 계산해주는 Wrap 클래스를 Composition 관계로 소유하고 있다.
