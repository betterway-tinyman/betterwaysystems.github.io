    Erase element.

    Removes from the Vector either a single element; position.

    This effectively reduces the container size by the number of element removed.

    Because Vectors use an Array as their underlying storage, erasing an element in position other than the end() causes the container to relocate all the elements after the segment erased to their new positions. This is generally an inefficient operation compared to the one performed for the same operation by other kinds of sequence containers (such as List).

      Iterator pointing to a single element to be removed from the Vector.
