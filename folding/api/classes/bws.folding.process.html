Class Process

공정 클래스.

Process는 개별 공정에 관한 모든 정보를 지니는 Facade 클래스이다.

현 공정이 사용하는 장비 Device 와, 조판에 사용될 Image 및 가격모형 PriceModels 객체를 직접 지닌다. 더불어 조판 배치정보 Placement 객체들을 Composition 관계로 지니고 있다.
