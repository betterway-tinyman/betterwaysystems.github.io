Folding

용어

도

색상코드의 갯수

  • ex) CMYK: 4도

면

출력에 사용될 종이면의 수량

  • 단면: 매당 1면
  • 양면: 매당 2면
  • 페이지 수 / 분할수량

가령 A4 사이즈 1,600페이지를 인쇄코자 하는데, 이를 A2에 양면 인쇄하여, A2 한 장 당 A4를 8장씩 담을 수 있다면, 이 때 매수는 200매가 된다.

  • 페이지 수: 1,600 페이지
  • 분할수량: 장당 8 페이지
  • 매수 = 페이지 수 / 분할수량 = 1,600페이지 / 8페이지 = 200매

판

도장과 비슷하다.

  • 판수 = 원본 면수 X 도
  • 판수 = 원본 페이지수 / 분할수량 X 도

작업비

판수 (면수) X 부수 (사본 수)

견적

용지

매수 x 용지면적 (제곱미터당)

인쇄

  1. 작업비: 7,000원 / 1,000면 X 도 (CMYK: 4도) - 계단형 변동비
  2. CTP 출력비: 판 X 도 X 7,000 원 - 변동비
  3. ex) 800페이지의 책 A를 CMYK로 2,000부 인쇄하려고 한다. 한 판당 8페이지를 넣어 총 100판이 생겼다.
    • 작업비: 2,800,000원
      • 단위: (100면 X 2,000부 / 1,000면) = 200 단위
      • 도: CMYK 4도
      • 작업비 = 7,000원 X 200단위 X 4도 = 5,600,000원
    • CTP 출력비: 140,000원
      • 원본면수: 800페이지 = 100면
      • 판수 = 원본면수 X 도수 = 100면 X 4도 = 400판
        • 매수 X 도수만큼의 판을 찍어야 한다.
      • 400판 X 7,000원 = 2,800,000원
  4. 양면일 때 행하는 돈땡은, 오로지 양면의 색상도수가 다를 때도 가능하되, 큰 색상도수의 비용을 사용해야 한다.
비고
  • 양단면 선택 가능
  • 여유 용지: (판) 면당 125 매

박

  1. 단위당 작업비: 25,000원 / 1,000면 - 변동비
  2. 한 판 동판비 (도장): 200원 / cm^2 - 변동비
  3. 한 판 박비 (잉크비): 1,300원 / m^2 - 변동비
비고
  • 양단면 선택 가능
  • 여유 용지: 면당 30 매

스코딕스

  1. 인쇄 단위당 작업비: 230원 / 1247.4cm^2
    • 1247.4cm^2: A3 1면
  2. 한 판 잉크비: 15원 / 12.474cm^2
    • 12.474cm^2: A3 면적대비 1% 잉크량

비고

  • 양단면 선택 가능
  • 여유 용지: 면당 30 매

톰슨

  1. 단위당 작업비 1.1. 고정비용: 30,000 원 / 1,000 매 1.2. 변동비용: 50원 / 매 1.3. 수식: 30,000 + 50 X (1,000 - x)
  2. 한 판 칼비 (도장, 판 only): 5,000원 / m
    • 추후 복잡도에 따라 가격이 변할 수 있다.
비고

라미네이팅

특이사항: 라미네이팅은 조판면이 아닌, 각 용지크기별로 단위를 매긴다.

  1. 작업비
    • 기본 요금 25,000원, 200면까지
    • 200면 초과는 라미네이팅 용지비 사용
  2. 필름 코팅비
    • 초과용지면적 (m2) x 200 원 / m2
    • 라미네이팅으로 코팅 가능한 종이는 크기에 한계가 있다
    • 이런 사용가능 종이 와 같은 정보는, 장치에 종속된다.
비고
  • 양단면 선택 가능
  • 여유 용지: 면당 30 매
