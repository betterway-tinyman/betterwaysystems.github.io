Folding

Folding 은 오프셋 견적을, 접지에 대한 (최적) 최소 비용을 도출해내는 솔루션이다.

  • GitHub Repository
  • API 문서
  • Class Diagram

빌드하기

# TypeScript Compiler 가 필히 설치되어있어야 한다.
npm install -g typescript

#------------
# 코어 연산부
#------------
# TypeScript 가 설치되었다면, 다음 명령어들을 수행한다.
# lib/folding-core.js 및 d.ts 가 생상될 것이다.
npm install --save
tsc -p ts/core.json 

#------------
# 데모 어플리케이션
#------------
# Demo Application 을 빌드하려거든, 다음 명령어를 수행한 후,
# ts/index.html 을 실행하면 된다.
tsc -p ts/tsconfig.json

용어

도

색상코드의 갯수

  • ex) CMYK: 4도

면

출력에 사용될 종이면의 수량

  • 단면: 매당 1면
  • 양면: 매당 2면
  • 페이지 수 / 분할수량

가령 A4 사이즈 1,600페이지를 인쇄코자 하는데, 이를 A2에 양면 인쇄하여, A2 한 장 당 A4를 8장씩 담을 수 있다면, 이 때 매수는 200매가 된다.

  • 페이지 수: 1,600 페이지
  • 분할수량: 장당 8 페이지
  • 매수 = 페이지 수 / 분할수량 = 1,600페이지 / 8페이지 = 200매

판

도장과 비슷하다.

  • 판수 = 원본 면수 X 도
  • 판수 = 원본 페이지수 / 분할수량 X 도

작업비

판수 (면수) X 부수 (사본 수)

가격모형

용지

매수 x 용지면적 (제곱미터당)

인쇄

  1. 작업비: 7,000원 / 1,000면 X 도 (CMYK: 4도) - 계단형 변동비
  2. CTP 출력비: 판 X 도 X 7,000 원 - 변동비
  3. ex) 800페이지의 책 A를 CMYK로 2,000부 인쇄하려고 한다. 한 판당 8페이지를 넣어 총 100판이 생겼다.
    • 작업비: 2,800,000원
      • 단위: (100면 X 2,000부 / 1,000면) = 200 단위
      • 도: CMYK 4도
      • 작업비 = 7,000원 X 200단위 X 4도 = 5,600,000원
    • CTP 출력비: 140,000원
      • 원본면수: 800페이지 = 100면
      • 판수 = 원본면수 X 도수 = 100면 X 4도 = 400판
        • 매수 X 도수만큼의 판을 찍어야 한다.
      • 400판 X 7,000원 = 2,800,000원
  4. 양면일 때 행하는 돈땡은, 오로지 양면의 색상도수가 다를 때도 가능하되, 큰 색상도수의 비용을 사용해야 한다.
비고
  • 양단면 선택 가능
  • 여유 용지: (판) 면당 125 매

박

  1. 단위당 작업비: 25,000 원 / 1,000 면 - 변동비
  2. 한 판 동판비 (도장): 200 원 / cm2 - 변동비
  3. 한 판 박비 (잉크비): 1,300 원 / m2 - 변동비
비고
  • 양단면 선택 가능
  • 여유 용지: 면당 30 매

스코딕스

  1. 피딩비: 230원 / 회1247.4cm^2
    • 1247.4cm^2: A3 1면
  2. 한 판 잉크비: 15원 / 12.474cm^2
    • 12.474cm^2: A3 면적대비 1% 잉크량

비고

  • 양단면 선택 가능
  • 여유 용지: 면당 30 매
  • 할증률이 들어갈 수도 있다 (출력 면수에 반비례한 할증률)

톰슨

  1. 단위당 작업비 1.1. 고정비용: 30,000 원 / 1,000 매 1.2. 변동비용: 50원 / 매 1.3. 수식: 30,000 + 50 X (1,000 - x)
  2. 한 판 칼비 (도장, 판 only): 5,000원 / m
    • 추후 복잡도에 따라 가격이 변할 수 있다.
비고
  • 여유 용지: 면당 30 매

라미네이팅

특이사항: 라미네이팅은 조판면이 아닌, 각 용지크기별로 단위를 매긴다.

  1. 작업비
    • 기본 요금 25,000원, 200면까지
    • 200면 초과는 필름 코팅비를 사용한다
  2. 필름 코팅비
    • 초과용지면적 (m2) x 200 원 / m2
    • 라미네이팅으로 코팅 가능한 종이는 크기에 한계가 있다
    • 이런 사용가능 종이 와 같은 정보는, 장치에 종속된다.
비고
  • 양단면 선택 가능
  • 여유 용지: 면당 30 매

특이사항

판걸이와 돈땡

판걸이는 현재의 조판면을 N배 확장하여, 더 많은 페이지를 배치하며, 돈땡은 조판면을 2배 확장하고 앞뒷면을 교차배치한다.

판걸이는 복수(N)부의 사본을 하나의 조판면에 찍는 확장법이고, 돈땡은 하나의 사본에 대한 확장법이다.

항목 판걸이 돈땡
배치 이미지 수의 확장 N 배 2배
배치 방식 순차 교차
배치되는 사본 수 N 부 1부

양-단면이 혼재하는 공정

각 모형별로 양-단면이 혼재하는 경우, 각 공정이 사용하는 판면수가 변할 수 있다 (단, 접지를 행하지 않을 때. 접지를 행하는 경우, 단면 인쇄 또한 양면 조판을 해야되기 때문에 해당사항 없음).

이 때, 인쇄가 양단면의 기준이 된다.

  • 인쇄가 단면이되, 그 외의 공정이 양면이면, 그 공정은 2배의 판면수를 쓰게 될 것이고,
  • 인쇄가 양면이되, 그 외의 공정이 단면이면, 그 공정은 절반의 판면수를 사용하게 된다.
