Interface IExternalServerClientArray

An interface for an ExternalSystemArray accepts external clients as a server and connects to IExternalServer as client, at the same time.

The easiest way to defining an IExternalServerClientArray who opens server, accepts external clients and connects to external servers is to extending one of below, who are derived from this interface IExternalServerClientArray. However, if you can't specify an ExternalSystemArray to be whether server or client or even can both them, then make a class (let's name it as BaseSystemArray) extending ExternalSystemArray and make a new class (now, I name it BaseServerClientArray) extending BaseSystemArray and implementing this interface IExternalServerClientArray. Define the BaseServerClientArray following those codes on below:

  • ExternalServerClientArray: View source code on GitHub
  • ParallelServerClientArray: View source code on GitHub
  • DistributedServerClientArray: View source code on GitHub
