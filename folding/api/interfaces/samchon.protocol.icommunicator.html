Interface ICommunicator

An interface taking full charge of network communication.

ICommunicator is an interface for communicator classes who take full charge of network communication with remote system, without reference to whether the remote system is a server or a client. Type of the ICommunicator is specified to IServerConnector and IClientDriver whether the remote system is a server (that I've to connect) or a client (a client connected to my server).

Whenever a replied message comes from the remote system, the message will be converted to an Invoke class and the Invoke object will be shifted to the listener's IProtocol.replyData() method.
