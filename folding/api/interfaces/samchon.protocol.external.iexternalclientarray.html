Interface IExternalClientArray

An interface for an ExternalSystemArray accepts external clients as a server.

The easiest way to defining an ExternalSystemArray who opens server and accepts external clients is to extending one of below, who are derived from this interface IExternalClientArray. However, if you can't specify an ExternalSystemArray to be whether server or client, then make a class (let's name it as BaseSystemArray) extending ExternalSystemArray and make a new class (now, I name it BaseClientArray) extending BaseSystemArray and implementing this interface IExternalClientArray. Define the BaseClientArray following those codes on below:

  • ExternalClientArray: View source code on GitHub
  • ParallelClientArray: View source code on GitHub
  • {@link DistributedClientArray}: View source code on GitHub
