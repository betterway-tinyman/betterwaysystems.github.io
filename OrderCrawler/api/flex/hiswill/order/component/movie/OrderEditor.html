handleChange()method 
protected function handleChange(event:Event):void

Change event handler; update Order.

All the events occured by editing items on the OrderEditor are dispatched into this method handleChange. It directly updates Order data whenever user handles UI items on this OrderEditor.

However, something to notice is that, the final data of the Order only be changed by clicking the merge button.
