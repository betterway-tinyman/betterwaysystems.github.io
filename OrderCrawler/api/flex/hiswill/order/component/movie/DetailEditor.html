Detail editor.

DetailEditor is an editor who can modify multiple Details at once.

Member data in the target Details dont't be changed until user clicks the OrderEditor.handleChange.

handleChange()method 
public function handleChange(event:Event):void

Change event handler; update Details.

All the events occured by editing items on the DetailEditor are dispatched into this method handleChange. It directly updates Details data whenever user handles UI items on this DetailEditor.

However, something to notice is that, the final data of the Details only be changed by clicking the OrderEditor.handleChange.
