A post-process.

PostProcess represents a post-process. It has process and value which mean an identifier of its post-process type and detailed value for the post-process.

Furthermore, PostProcess not only acts a role of entity, but also it participates in UI-editor by supporting a factory method toFormItemArray() who returns a FormItems who can edit this PostProcess directly.
