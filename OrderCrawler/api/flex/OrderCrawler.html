sendLoad()method 
protected function sendLoad(event:MouseEvent):void

Send load instruction to C++ server.

When the load instruction was sent, The C++ server starts crawling from contractors' homepages and constructs Order instances. After the crawling was done, new message setContractorArray will come again from the C++ server.

The setContractorArray changes all entities and UI-boundaries what you've seen. As that reason, the program asks you whether to send the load instruction when you're editing an Order or Detail instance.

event:MouseEvent
