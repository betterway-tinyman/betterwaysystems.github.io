상세한 설명

후가공 엔티티

단일 후가공 프로세스를 대표하는 엔티티로써, 자신의 타입을 의미하는 프로세스 코드와, 해당 타입 내 상세 값을 의미하는 밸류 코드를 가진다. PostProcess는 위와 같은 기본 엔티티의 역할에 더불어, 위 프로세스 및 밸류 코드의 유효성을 검증하고, DB와의 입출력을 담당하는 역할을 겸한다.
