외부 계약사 엔티티

하나의 외부 계약사를 대표하는 엔티티로써 Order의 배열 역할을 겸하며, 해당 외부 계약사의 주문기록 일체에 대한 크롤링과 DB I/O 프로세스 일체를 시작하는 메소드로 담은, 일종의 Facade 클래스이다.

  • ISQLEntity: http://samchon.github.io/framework/api/cpp/d7/d56/classsamchon_1_1protocol_1_1ISQLEntity.html
작성자
Jeongho Nam http://samchon.org

현재의 주문목록 엔티티 일체를 DB에 기록한다.

T-SQL의 insert buffer 크기 한계를 넘지 않기 위하여, 주문기록 엔터티를 10개 단위로 끊어 기록한다. 또한, 신규 주문기록의 경우, DB에서 생성된 새 식별자 번호, uid를 받아 설정해준다.
