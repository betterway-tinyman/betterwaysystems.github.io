bool hiswill::order::allowed::Product::enableCustomScale_
private

사용자가 직접 크기를 정할 수 있는 지 여부

enableCustomScale이 true이면, 해당 제품에 제공되는 사이즈 템플릿 (sizeArray) 과 별개로 사용자가 직접 제품의 재단 크기를, 지정된 한계값을 벗어나지 않는 범위(minWidth ~ maxWidth, minHeight ~ maxHeight)에서 입력할 수 있다.
