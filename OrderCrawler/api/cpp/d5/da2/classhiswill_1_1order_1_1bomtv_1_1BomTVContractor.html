외부 계약사, BomTV 엔티티

[Inherited] Contractor
하나의 외부 계약사를 대표하는 엔티티로써 Order의 배열 역할을 겸하며, 해당 외부 계약사의 주문기록 일체에 대한 크롤링과 DB I/O 프로세스 일체를 시작하는 메소드로 담은, 일종의 Facade 클래스이다.
