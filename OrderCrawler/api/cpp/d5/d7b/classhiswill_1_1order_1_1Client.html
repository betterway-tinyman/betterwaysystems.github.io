상세한 설명

클라이언트 (드라이버).

Client 는 Flex 클라이언트와의 소켓 통신 및 메시지 핸들링을 담당하는 클래스이다.

Flex 클라이언트가 서버에 접속을 하면, Server 에서 현재의 Client 객체를 생성하고, 현재의 Client 객체가 Flex 클라이언트로부터 받은 메시지를 판별하여 (#replyData) 클라이언트 내에서 처리될 작업이면, Client 내에서 처리하고, 서버 단위에서 이루어져야 할 작업이면 Server::replyData 로 메시지를 넘긴다 (Chain of responsibility).

  • IClient: http://samchon.github.io/framework/api/cpp/d3/dc8/classsamchon_1_1protocol_1_1IClient.html
